# MarketMind Solutions — Website

Official website for MarketMind Solutions, a mobile clinic operations company.

**Live site:** https://mmsolutionsmobile.com

## Stack
- Plain HTML / CSS / JavaScript — no framework
- Hosted on Cloudflare Pages (free tier)
- Contact form via Formspree
- Domain registered via GoDaddy

## Pages
- `index.html` — Home
- `about.html` — About
- `services.html` — Services
- `careers.html` — Careers
- `contact.html` — Contact
- `styles.css` — Shared stylesheet (all 5 pages)

## Updating the Site
Edit files directly in GitHub using the pencil icon.
Cloudflare Pages auto-deploys within ~60 seconds of any commit.
